### 1. Create a file with .txt extension
```bash
sudo touch /home/demo.txt
```
### 2. Change the permission set of that file
```bash
sudo chmod 764 /home/demo.txt
```
### 3. Permission breakdown
Owner (7) → Read (r) + Write (w) + Execute (x)
Group (6) → Read (r) + Write (w)
Others (4) → Read (r)
